1. Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>(sem semântica)

Enquanto nessa frase temos um termo em destaque utlizando a tag <strong>

2. Itálico / Ênfase

Nesta frase temos um termo em itálico, usando a tag <i> (sem semântica)

Nesta frase temos um termo em ênfase utilizando a tag <em>

3. Texto Marcado

Podemos também criar um texto marcado utilizando a tag mark

4. Texto Grande e Texto Pequeno

Utlizando a tag big. E utilizando a tag small. Ex01 Gato e Cachorro

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

5. Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
