Grupo de Tags

As tags de agrupamento são usadas para agrupar outros elementos HTML. Elas não têm um significado semântico específico, mas ajudam a organizar o conteúdo da página.

As tags de agrupamento mais comuns são:

  • <div>: Usada para agrupar elementos em um bloco.
  • <span>: Usada para agrupar elementos em linha.
  • <section>: Usada para agrupar seções de conteúdo.
  • <article>: Usada para agrupar artigos independentes.
  • <aside>: Usada para agrupar conteúdo relacionado, mas não essencial.
